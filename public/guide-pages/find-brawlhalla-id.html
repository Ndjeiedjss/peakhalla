How to Find Your Brawlhalla ID

Find and use a Brawlhalla ID to open a player profile even when the display name changes.

Overview

A Brawlhalla ID is a stable numeric identifier for an account. It is more reliable than a display name because names can change.

Why the ID matters

PeakHalla can use the numeric ID to connect saved aliases, ranked snapshots, clan data and profile history to the same account.

Search by ID

Paste the numeric ID into PeakHalla’s player search. If the account has been seen before, its saved profile can open immediately.

Use it on PeakHalla

Search a player or open the matching ranked leaderboard to see the information in a live profile. PeakHalla keeps saved data ready for fast loading and checks current sources in the background.
